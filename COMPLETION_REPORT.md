## 💎 **Technical Specifications**

### 🏗️ **Architecture**
- **Protocol**: Model Context Protocol (MCP) 1.0
- **Runtime**: Node.js 18+
- **Language**: TypeScript 5.8+
- **Authentication**: HTTP Basic Auth
- **Transport**: stdio (standard MCP)
- **Distribution**: npm global package
